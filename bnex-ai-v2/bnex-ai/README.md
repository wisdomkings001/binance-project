## Setup

### 1. Add Your API Key
Open `src/App.jsx` and replace line 3:
```
const API_KEY = "YOUR_ANTHROPIC_API_KEY_HERE";
```
Get your key at: https://console.anthropic.com

### 2. Install & Run
```bash
npm install
npm start
```

### 3. Build for Netlify
```bash
npm run build
```
Drag the `build/` folder to netlify.com
